SponsorPay Challenge
====================

1. Clone the respository in your local machine.

2. Install bundle gem and run bundle install command:

	```
	$ gem install bundle
	$ bundle install
	```
	
3. And run with:

	```
	$ ruby myapp.rb
	```

4. Open ```http://localhost:4567``` in your web browser


Run specs (tests)
-----------------

Run with:

```
$ rake spec
```

or just:

```
$ rake
```
